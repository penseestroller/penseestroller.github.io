Visual Studio 配置注视模板

发表于 2019-04-22 分类于 开发工具 Valine： 本文字数： 748 阅读时长 ≈ 1 分钟

用 VS 开发程序的时候，我们往往需要个性化自己的注释，而这些注释是可以模板化的。下面就是具体配置我们的注释模板。

我们以VS2017/VS2019为例子来进行相应配置。
在路径
C:\Program Files (x86)\Microsoft Visual Studio\2017\Enterprise\Common7\IDE\ItemTemplates\CSharp\Code\2052 下有名为 Class、CodeFile、Interface、WebClass几个文件夹，分别修改文件夹中的 .cs文件，添加自己的注释信息：

附：注释模版本中的参数说明：

参数 说明
clrversion 当前系统CLR版本号
guid [1-10] 生成全局唯一标识符,可以生成10个 (例如:guid1)
itemname 设置容器CPU权重，在CPU共享场景使用
machinename 当前机器的名称
registeredorganization 注册的公司名称
rootnamespace 命名空间
safeitemname 保存的文件名
time 当前系统时间,格式:DD/MM/YYYY 00:00:00.
usernamnam 用户名
userdomain 用户所在的域
year 当前系统时间 YYYY
